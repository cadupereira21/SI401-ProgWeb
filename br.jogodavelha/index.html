COMO JOGAR?


1. Clique em um dos quadrados para adicionar um elemento!

2. O jogo é dividido por turnos, o primeiro é sempre do elemento círculo!

3. Quando um jogador vencer, o tabuleiro será automaticamente resetado!

4. Em caso de velha, o tabuleiro só será resetado quando todas as casas estiverem cheias!

5. Caso note que o jogo terminará em velha, o botão 'Reiniciar' resetará o tabuleiro sem ninguém pontuar!

OBS: O botão 'Jogar' te levará para a tela de jogo, porém NÃO é necessário para iniciá-lo!
